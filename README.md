# CLI Timer

Simple timer and stopwatch you can run from anywhere.

## Install
```bash
sudo apt update && sudo apt install pipx -y && pipx ensurepath && pipx install git+https://github.com/Timmy6942025/cli-timer


Usage
bash
Copy
Edit
timer 5min 30sec
stopwatch
Controls
p = Pause / Resume

r = Reset

e or Ctrl+C = Exit

+ = Increase size

- = Decrease size
